# Pong (advanced project)

## Installation

1. Install mcb32tools from the Github respository: https://github.com/is1200-example-projects/mcb32tools/releases/
2. Build and run according to instructions in README file of toolchain

## Controls

Button 4 is up
Button 3 is down
Button 2 is continue in menu and up for player 2 in PvP mode
Button 1 is "back to main menu" in menu and down for player 2 in PvP mode
